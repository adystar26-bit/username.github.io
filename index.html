⚡ Tips Cepat

1) Ganti DNS

DNS cepat sering bikin browsing & buka aplikasi lebih responsif di jaringan padat.

Cloudflare 1.1.1.1
Google 8.8.8.8
  • Android: Setelan → Jaringan → DNS Pribadi → isi 1dot1dot1dot1.cloudflare-dns.com

  • Simpan APN baru, lalu restart data seluler.

3) Kunci LTE Only (Android)

Menghindari pindah-pindah ke 3G saat sinyal lemah. Gunakan dengan bijak.

Dial Code *#*#4636#*#*
  • Buka → Phone Information → Set Preferred Network ke LTE only
  • Jika tidak ada sinyal telepon, kembalikan ke LTE/WCDMA.

4) Hemat Bandwidth

Matikan auto-update & batasi data latar.

  • Play Store → Setelan → Update otomatis → Hanya via Wi-Fi
  • Android: Setelan → Penggunaan Data → Penghemat Data ON
  • Tutup aplikasi berat di latar (TikTok, IG, YouTube autoplay)

🌐 Hosting di GitHub Pages

Halaman ini bisa kamu host gratis. Cocok buat simpan toolkit pribadi.

  1. Buat repository

    Login GitHub → New repo → nama: username.github.io (ganti username).

  2. Upload file

    Upload file HTML ini sebagai index.html ke branch main.

  3. Aktifkan Pages

    Settings → Pages → Source: Deploy from a branch → Branch: main / root → Save. Akses di https://username.github.io
